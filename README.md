## 📌 Descrição

Este repositório contém a implementação do jogo Movie To Move, desenvolvido em Portugol. O projeto simula um jogo de tabuleiro para dois jogadores, no qual o avanço ocorre por meio do sorteio de dados e eventos especiais inspirados em frases marcantes do cinema.

O sistema utiliza menu interativo, placar acumulativo e estruturas condicionais que tornam cada partida dinâmica e imprevisível.

---

## 📋 Estrutura do Projeto

O projeto foi organizado seguindo os seguintes critérios:

Menu principal com opções de iniciar jogo, visualizar placar ou finalizar.

Sistema de turnos alternados entre dois jogadores.

Geração de números aleatórios simulando lançamento de dados.

Casas especiais com eventos personalizados (avanço, recuo, troca de posição, desafios).

Controle de vitória ao atingir a casa 20.

Sistema de placar acumulativo durante a execução do programa.
